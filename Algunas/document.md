# Residuo o Modulo

### El residuo es cuando un numero al dividirse por otro su division es exacta o no.

---

### Ejemplo:
_Como el numero **2** al dividirse por **4** su residuo es **0.5** osea **.5** o el **2** al dividirse por **3** su residuo es **0.67** osea .67_

### Como nos damos cuenta el residuo es la parte decimal del resultado de una divicion.

``` java

public void Residuo(float a, float b){

     if(a % b == 0) System.out.println("La divicion entre a y b su residuo es 0");
     else System.out.println("La divicion entre a y b su residuo no es 0");

}

```

_En programacion para saber el residuo usamos en porcetaje o modulo **%**_
